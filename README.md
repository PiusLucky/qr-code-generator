

## Description
A simple QR code that generates 10 random movies.

## Deployed link
https://smart-qr-generator.onrender.com/

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run  dev

# production mode
$ yarn run start:prod
```
## Stack
- Prisma ORM
- PostgreSQL
- Nestjs
- Typescript
